# User Stories

## 1
As a user  
I want to register an account  
So that I can access the platform  

## 2
As a user  
I want to log in securely  
So that I can manage my profile  

## 3
As a user  
I want to view available items  
So that I can find useful things  

## 4
As a user  
I want to search items by category  
So that I can quickly find relevant items  

## 5
As a user  
I want to view item details  
So that I can decide before requesting  

## 6
As a user  
I want to post items  
So that I can give away unused things  

## 7
As a user  
I want to update my profile  
So that my information stays accurate  

## 8
As a user  
I want to delete my account  
So that I can control my data
